# LinkList 类的 `length` 方法是通过哪种方式计算链表的长度?

---

### 🧩 题目选项

A. 统计链表中数据不为 None 的节点个数
B. 从链表头节点开始，依次遍历每个节点并计数
C. 检查链表中每个节点的 `next` 属性，统计不为 None 的次数
D. 根据链表头节点的 `data` 属性值来确定长度

---

### 💡 解题思路

链表的长度通常是通过遍历整个链表来计算的：

1. 从链表的头节点（`head`）开始；
2. 使用一个临时指针（`cur`）；
3. 每访问一个节点，计数器加一；
4. 当指针到达 `None`（链表尾部）时，停止遍历；
5. 返回计数结果。

---

### ✅ 代码示例

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkList:
    def __init__(self):
        self.head = None

    def length(self):
        count = 0
        cur = self.head
        while cur:
            count += 1
            cur = cur.next
        return count
```

在这个过程中，程序是 **从头节点开始，依次遍历每个节点并计数**。

---

### 🧠 选项分析

| 选项 | 含义                       | 是否正确 | 说明                                       |
| ---- | -------------------------- | -------- | ------------------------------------------ |
| A    | 统计数据不为 None 的节点   | ❌        | **数据可能为 None，但节点依然存在**        |
| B    | 从头节点开始依次遍历并计数 | ✅        | 与标准实现完全一致                         |
| C    | 检查 next 属性不为 None    | ❌        | **最后一个节点的 next 是 None，会少算 1 ** |
| D    | 根据 data 属性值确定       | ❌        | data 与链表长度无关                        |

---

### ✅ 正确答案

**B. 从链表头节点开始，依次遍历每个节点并计数**
