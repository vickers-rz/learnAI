一、列表和元组

（一）列表

- 列表的格式

``` python
[数据1, 数据2, 数据3]
```

- 常用操作方法（查、增、删、改、判断）

  查：

  - 直接通过下标查询

  - index()：返回指定数据所在位置的下标 。

  ``` python
  列表序列.index(数据, 开始位置下标, 结束位置下标)
  ```

  - len()
  - 直接通过下标查询
  - append()
  - pop()
  - remove()

- 列表嵌套

``` python
name_list = [['小明', '小红', '小绿'], ['Tom', 'Lily', 'Rose'], ['张三', '李四', '王五']]
name_list[2][1]
```

（二）元组

- 定义元组

``` python
t1 = (10, 20, 30)

t2 = (10,)
```

- 常用操作方法
  - index()
  - len()

二、集合和字典

（一）集合：

- 创建集合

  - 有数据集合

  ``` python
  s1 = {数据1, 数据2, ...}
  ```

  - 无数据集合

  ``` python
  s1 = set()
  ```

- 常见操作

  - 增加数据
    - add()
    - update()
  - 删除数据
    - remove()
    - discard()

（二）字典：

- 定义字典

``` python
dict1 = {'name': 'Python', 'age': 30}

dict2 = {}

dict3 = dict()
```

- 常见操作
  - 增/改

``` python
字典序列[key] = 值
```

- 查找
  - 字典序列[key]
  - keys()
  - values()
  - items()
