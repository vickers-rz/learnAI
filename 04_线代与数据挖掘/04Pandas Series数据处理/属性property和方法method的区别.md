 **“属性（property）”** 和 **“方法（method）”** 是 Python 面向对象编程中非常核心的两个概念。我们来一步步讲清楚👇

------

## 🧩 一、属性（Property）

**属性** 是对象中存储的数据或特征，**不用加括号** 就能访问。

比如一个对象就像“人”，属性就是这个人的特征，比如名字、年龄。

### 🧠 举例：

```
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

p = Person("Alice", 25)
print(p.name)  # 属性访问，不加括号
print(p.age)
```

输出：

```
Alice
25
```

这里的 `name` 和 `age` 就是属性。

在你的截图中：

```
data.loc
```

`loc` 就是一个 **属性**。
它返回一个 **“定位器对象”**（`.LocIndexer`），用来让你后面再执行定位操作，比如：

```
data.loc[0, 'year']
```

👉 获取第 0 行、`year` 列的数据。

------

## ⚙️ 二、方法（Method）

**方法** 是对象能执行的“动作”或“操作”，**调用时必须加括号**。

比如“人”能做的事是方法，比如走路、说话。

### 🧠 举例：

```
class Person:
    def speak(self):
        print("Hello!")

p = Person()
p.speak()  # 方法调用，需要括号
```

输出：

```
Hello!
```

在你的例子中：

```
data.head()
```

`head()` 就是一个 **方法**，它的作用是“显示前几行数据”。

------

## 🔍 对比总结

| 对象特性   | 属性 (Property)                          | 方法 (Method)                                   |
| ---------- | ---------------------------------------- | ----------------------------------------------- |
| 用途       | 保存或表示数据                           | 执行操作或功能                                  |
| 是否加括号 | ❌ 不加括号                               | ✅ 要加括号                                      |
| 例子       | `data.shape`, `data.loc`, `data.columns` | `data.head()`, `data.describe()`, `data.info()` |
| 类比       | 人的“身高”“年龄”                         | 人的“走路”“说话”                                |

------

💬 所以你在截图里看到的：

```
@property
def loc(self)
```

说明 `loc` 是一个 **属性**（用 `@property` 修饰），
而像 `head()` 则是一个 **方法**。
